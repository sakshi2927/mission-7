# prompt.md

## AI Prompts Used During Development

This file documents the basic prompts I used with an AI assistant while building the **Registration Wizard** project.  
The goal was to get light guidance and clarification, not full solutions or copy-paste code.

---

### 1. Project Planning

**Prompt:**
> I want to build a multi-step registration wizard in React.  
> Can you help me break this into steps and components?

**Why I Asked This:**  
To understand how to structure the project and split the UI into steps.

---

### 2. State Management

**Prompt:**
> How can I manage form state across multiple steps in React so data is not lost when navigating back and forth?

**Why I Asked This:**  
To learn how to store form data in a parent component and pass it down as props.

---

### 3. Validation Basics

**Prompt:**
> How can I add simple real-time validation for email and password fields in React?

**Why I Asked This:**  
To understand basic validation logic using conditions and regex.

---

### 4. UX Improvements

**Prompt:**
> How can I disable the Next button until all required fields in the current step are valid?

**Why I Asked This:**  
To improve user experience and prevent moving forward with invalid input.

---

### 5. Styling & Layout

**Prompt:**
> How do I center my form wizard in the middle of the screen using CSS?

**Why I Asked This:**  
To improve layout and presentation of the UI.
